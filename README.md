# Staylytics

Staylytics is an interactive analytics web application for exploring the Airbnb rental market in London.

It helps users filter, analyse, and interpret listing data through dynamic visualisations, pricing analytics, and rule-based market insights.

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- Recharts

## Features

- Overview dashboard for London Airbnb listings
- Prices page with pricing KPIs, distributions, rankings, and insights
- Borough, room type, and price filters
- Rule-based AI-style summaries
- Responsive SaaS-style dashboard layout

## Run Locally

```bash
npm install
npm run dev
```
